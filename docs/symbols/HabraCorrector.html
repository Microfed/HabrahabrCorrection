Добавляет событие для сочетания клавиш Ctrl+Enter. При срабатывании показывает диалоговое окно для отправки сообщения об ошибке.
alertWhenFail(response)
Проверяет ответ севера. Если ответ содержит информацию об ошибке, сообщает эту информацию пользователю посредством alert()

response
Ответ на запрос к серверу
